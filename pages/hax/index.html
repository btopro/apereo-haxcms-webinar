What is HAX?

HAX is short for Headless Authoring eXperience . It is a solution implemented via the web component standard which seeks to create the ultimate authoring solution for the web and implement it in any system (hence headless).

HAX is born out of ELMS:LN and while ELMS:LN implements HAX, HAX works in many modalities.

This presentation is a sampling of the documentation and resources available on HAXTheWeb.org
